    .. rst-class:: h1 

      **About Netket**

    .. rst-class:: h2 

      NetKet's mission is to provide researchers with state-of-the-art machine learning techniques for many-body quantum systems.


.. container:: centered-bubble

    .. container:: jumbotron-fluid centered-bubble-header
      
        .. rst-class:: text-muted

                There is a fast-growing interest in neural-network-based approaches to solve the quantum many-body problem. NetKet is the first open-source platform for collaborative developments in the field.
